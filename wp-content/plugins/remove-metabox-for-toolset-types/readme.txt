Remove Front-end display metabox from Toolset Types.

== Description ==

This plugin remove the Front-end display metabox that Toolset Types plugin includes by default.

This plugin will using a filter and return nothing (null) so this section will not be displayed anymore. Do not install this plugin if you rely on the Toolset Types plugin functions.

If you do not know what this means you should probably not install this plugin. If you are using a plugin such as Toolset Types plugin to manage your custom fields don't worry - this should be unaffected.

This plugin was created by SuitPress - http://suitpress.com

== Installation ==

Upload the this plugin to your blog, Activate it.

e.g.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
